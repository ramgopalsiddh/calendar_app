# Calendar App

### Setup Instructions

1. **Backend (NestJS)**:
   - Navigate to `backend/`.
   - Install dependencies: `npm install`
   - Start the server: `npm run start`
   - Server runs on `http://localhost:3000`

2. **Frontend (React)**:
   - Navigate to `frontend/`.
   - Install dependencies: `npm install`
   - Start the app: `npm start`
   - App runs on `http://localhost:3001`
